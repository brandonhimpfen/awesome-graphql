> A curated list of GraphQL frameworks, servers, clients, tooling, gateways, schema management platforms, educational resources, and community projects.

GraphQL has become one of the most widely adopted technologies for building flexible APIs and modern application architectures. It enables developers to efficiently query and manage data across web, mobile, cloud, and distributed systems.

This list is intended for software developers, API architects, DevOps engineers, platform teams, technical leaders, and students working with GraphQL technologies.

## Servers & Frameworks

Frameworks and server implementations for building GraphQL APIs.

- [GraphQL.js](https://github.com/graphql/graphql-js) — Reference implementation of GraphQL for JavaScript.
- [Apollo Server](https://www.apollographql.com/docs/apollo-server/) — Popular GraphQL server framework with ecosystem integrations.
- [GraphQL Yoga](https://the-guild.dev/graphql/yoga-server) — Fully featured GraphQL server focused on standards compliance.
- [Mercurius](https://github.com/mercurius-js/mercurius) — Fast GraphQL adapter for Fastify.
- [NestJS GraphQL](https://docs.nestjs.com/graphql/quick-start) — GraphQL integration for NestJS applications.
- [Hot Chocolate](https://chillicream.com/docs/hotchocolate/) — GraphQL server framework for .NET.
- [Graphene](https://graphene-python.org/) — GraphQL framework for Python.
- [Strawberry](https://strawberry.rocks/) — Python GraphQL library using modern type annotations.
- [Ariadne](https://ariadnegraphql.org/) — Schema-first GraphQL server library for Python.
- [Lighthouse](https://lighthouse-php.com/) — GraphQL server for Laravel and PHP applications.

## Clients

Libraries and SDKs for consuming GraphQL APIs.

- [Apollo Client](https://www.apollographql.com/docs/react/) — Feature-rich GraphQL client for JavaScript and React.
- [Relay](https://relay.dev/) — GraphQL framework developed by Meta for React applications.
- [urql](https://github.com/urql-graphql/urql) — Lightweight and extensible GraphQL client.
- [graphql-request](https://github.com/graffle-js/graffle) — Minimal GraphQL client focused on simplicity.
- [Apollo Kotlin](https://www.apollographql.com/docs/kotlin/) — GraphQL client for Android and Kotlin applications.
- [Apollo iOS](https://www.apollographql.com/docs/ios/) — GraphQL client for iOS development.

## Federation & Gateways

Tools for managing GraphQL across distributed services and microservice environments.

- [Apollo Federation](https://www.apollographql.com/federation/) — Framework for composing distributed GraphQL services.
- [Apollo Router](https://www.apollographql.com/docs/router/) — High-performance GraphQL router for federated architectures.
- [GraphQL Mesh](https://the-guild.dev/graphql/mesh) — Unified GraphQL gateway for multiple data sources.
- [Hive Gateway](https://the-guild.dev/graphql/hive/gateway) — Open-source GraphQL gateway platform.
- [Hasura DDN](https://hasura.io/) — Data delivery network supporting federated GraphQL architectures.

## Schema Management & Registries

Platforms and tools for schema governance, collaboration, and observability.

- [Apollo GraphOS](https://www.apollographql.com/graphos/) — Platform for GraphQL schema management and operations.
- [GraphQL Hive](https://the-guild.dev/graphql/hive) — Open-source GraphQL registry and observability platform.
- [GraphQL Inspector](https://github.com/graphql-hive/graphql-inspector) — Schema validation and change detection tool.
- [Apollo Rover](https://www.apollographql.com/docs/rover/) — CLI for GraphOS and schema workflows.

## Developer Tools

Utilities that support GraphQL development and exploration.

- [GraphiQL](https://github.com/graphql/graphiql) — Interactive GraphQL IDE and query explorer.
- [Apollo Sandbox](https://www.apollographql.com/docs/graphos/explorer/) — Browser-based GraphQL API explorer.
- [Altair GraphQL Client](https://altairgraphql.dev/) — Cross-platform GraphQL client and IDE.
- [Insomnia](https://insomnia.rest/) — API client supporting GraphQL development workflows.
- [Postman](https://www.postman.com/) — API platform with GraphQL support.
- [GraphQL Voyager](https://github.com/APIs-guru/graphql-voyager) — Visual exploration tool for GraphQL schemas.

## Testing & Mocking

Resources for validating and testing GraphQL APIs.

- [Mock Service Worker](https://mswjs.io/) — API mocking framework supporting GraphQL operations.
- [GraphQL Faker](https://github.com/APIs-guru/graphql-faker) — Mock GraphQL server generator.
- [Jest](https://jestjs.io/) — JavaScript testing framework commonly used with GraphQL projects.
- [Playwright](https://playwright.dev/) — End-to-end testing framework for GraphQL-powered applications.

## Code Generation

Tools that automate client and server development.

- [GraphQL Code Generator](https://the-guild.dev/graphql/codegen) — Generates typed clients, SDKs, and schema artifacts.
- [Apollo Kotlin Codegen](https://www.apollographql.com/docs/kotlin/) — Generates strongly typed Kotlin models.
- [Relay Compiler](https://relay.dev/docs/) — Generates artifacts for Relay applications.

Resources focused on securing GraphQL APIs.
